A. Sidebar a Sinistra (Default)

Tutti i controlli, i profili e le proprietà sono accumulati a sinistra. Il canvas sta a destra.

B. Right Inspector (Stile Figma)

Dock di strumenti sottile a sinistra. Canvas al centro. Proprietà ed elementi a destra (vicino all'area di disegno).

C. Pannelli Divisi (Doppia Sidebar)

Pannello sinistro per Dimensioni/Formati. Pannello destro per Proprietà dell'elemento attivo. Canvas al centro.

D. Pannelli Fluttuanti (Massimo Spazio)

La griglia e il canvas riempiono lo schermo. I pannelli fluttuano sopra l'area di disegno in trasparenza.

Analisi dei Layout

Ogni schema ha precisi impatti sull'usabilità all'interno di Grist (che è spesso incorporato come Iframe).

Layout Vantaggi Svantaggi
A. Left Sidebar Standard Tutto raggruppato in un unico posto. Clutter Molto scrolling verticale; allontana l'inspector dal canvas.
B. Right Inspector Ottimale Separa la barra strumenti dalle proprietà (stile Figma/Canva). Ergonomico. Larghezza Richiede almeno 700px di larghezza totale della finestra.
C. Split-Pane Zero Scroll I profili a sinistra e le proprietà a destra eliminano lo scrolling. Spazio Il canvas centrale risulta compresso se lo schermo è stretto.
D. Floating Panels Immersivo Massimizza l'area visiva del canvas; aspetto moderno ed elegante. Overlap I pannelli possono coprire gli elementi dell'etichetta.

Integrazione CSS

Il codice CSS di layout per trasformare il tuo `#main-content` attuale in un **Right Inspector (Figma-style)**:

/* 1. Struttura del Main */
#main-content {
    display: flex;
    flex-direction: row;
    overflow: hidden;
}

/* 2. Dock Strumenti Sottile a Sinistra */
#toolbar-vertical {
    width: 52px;
    background: #1e293b;
    border-right: 1px solid #334155;
    display: flex;
    flex-direction: column;
    align-items: center;
    gap: 12px;
    padding: 12px 0;
}

/* 3. Area Disegno Centrale */
#canvas-container {
    flex-grow: 1;
    overflow: auto;
}

/* 4. Inspector Proprietà a Destra */
#properties-inspector {
    width: 290px;
    background: #1e293b;
    border-left: 1px solid #334155;
}
