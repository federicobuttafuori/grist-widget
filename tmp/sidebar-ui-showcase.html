Prototipi di Layout e Componenti V4

Seleziona i prototipi per testarli interattivamente

1. Riorganizza Posizione

Scegli come disporre i pannelli principali all'interno del simulatore al centro.

A. Sidebar a Sinistra (Standard)

Tutti i controlli, i profili e le proprietà sono accumulati a sinistra. Il canvas sta a destra.

B. Right Inspector (Stile Figma)

Dock di strumenti a sinistra. Canvas al centro. Proprietà ed elementi a destra.

C. Pannelli Divisi (Doppia Sidebar)

Sinistra per Dimensioni/Formati. Destra per Proprietà. Canvas al centro.

D. Pannelli Fluttuanti (Massimo Spazio)

I pannelli di modifica fluttuano sopra l'area di disegno in trasparenza.

2. Prototipi Import/Export

Scegli come visualizzare l'area di salvataggio. Cambierà il blocco "Importa/Esporta" nella sidebar simulata.

1. Griglia con Scorciatoie

Griglia 2x2 compatta con badge che mostrano le scorciatoie da tastiera (es: Ctrl+E).

2. Tabbed File Hub + Dropzone

Tab divisi per Esporta e Importa. L'importazione ha un'area drag-and-drop interattiva.

3. Split Button Dropdown

Pulsante principale "Esporta" con freccia dropdown per scegliere il formato di salvataggio.

4. Card Informativa Integrata

Contenuto racchiuso in una card in risalto con brevi descrizioni dei formati di salvataggio.

3. Stile Padding & Bordo

Scegli come visualizzare le impostazioni del bordo e del padding negli elementi per confrontare l'ergonomia.

1. Figma Style (Compatto)

Controlli unificati con toggle d'espansione. Il colore e lo spessore del bordo stanno sulla stessa riga.

2. Classico (Verticale)

Pulsanti e input elencati verticalmente uno sotto l'altro. Tante righe di opzioni e molto scrolling.

Layout Vantaggi Svantaggi

Integrazione CSS

Il codice CSS di layout per trasformare il tuo `#main-content` attuale in un **Right Inspector (Figma-style)**:

/* 1. Struttura del Main */
#main-content {
    display: flex;
    flex-direction: row;
    overflow: hidden;
}

/* 2. Dock Strumenti Sottile a Sinistra */
#toolbar-vertical {
    width: 52px;
    background: #1e293b;
    border-right: 1px solid #334155;
    display: flex;
    flex-direction: column;
    align-items: center;
    gap: 12px;
    padding: 12px 0;
}

/* 3. Area Disegno Centrale */
#canvas-container {
    flex-grow: 1;
    overflow: auto;
}

/* 4. Inspector Proprietà a Destra */
#properties-inspector {
    width: 290px;
    background: #1e293b;
    border-left: 1px solid #334155;
}

Confronto Salva/Carica

Analizziamo i 4 prototipi per la sezione di Import/Export delle etichette:

OPZIONE 1 Griglia con Scorciatoie

Conserva la logica originaria a 4 bottoni ma li organizza in un blocco simmetrico 2x2. Visualizza la scorciatoia Ctrl+E per educare l'utente. **Voto Usabilità: 8/10**.

OPZIONE 2 Tabbed File Hub + Dropzone

Divide le azioni in tab. Il tab "Carica" offre un'area interattiva dashed drag-and-drop per importare i file. Riduce lo stress visivo. **Voto Usabilità: 9/10**.

OPZIONE 3 Split Button (Figma Style)

Raggruppa le varie esportazioni (solo Layout, Snapshot completo, o PNG) in un unico menu a discesa compatto. **Voto Usabilità: 8.5/10**.

OPZIONE 4 Card Informativa Integrata

Racchiude i pulsanti dentro un pannello azzurrato con brevi spiegazioni. Guida gli utenti meno esperti. **Voto Usabilità: 7.5/10**.

Ergonomia di Padding & Bordo

Confronto sull'usabilità per la configurazione di margini interni e contorni:

FIGMA STYLE Controlli Unificati e Collassabili

Padding: Un singolo campo numerico per il padding globale. Cliccando sull'icona di espansione, si aprono 4 mini-input per i lati singoli in orizzontale. Riduce il disordine.

Bordo: Colore e spessore condividono la stessa riga. I lati sono selezionabili tramite una barra segmentata compatta. Angoli configurabili con un input globale ed espansione a 4 angoli.

CLASSICO Elencazione Verticale

Mostra tutti i campi (Padding Top, Right, Bottom, Left) come righe singole fin dall'inizio. Costringe l'utente a scrollare continuamente la sidebar anche se vuole solo impostare un bordo semplice.

Integrazione CSS & HTML

Come ristrutturare la sezione del bordo in `index.html` per allineare colore e spessore:

<!-- Row Colore e Spessore affiancati -->
<div class="border-row-inline" style="display: grid; grid-template-columns: 1fr 80px; gap: 8px;">
    <div class="color-picker-wrapper">
        <input type="color" data-prop="props.borderColor">
    </div>
    <div class="thickness-wrapper">
        <input type="number" data-prop="props.borderWidth">
    </div>
</div>
